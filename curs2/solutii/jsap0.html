Dacă cuvântul cheie var este utilizat în cadrul unei funcții sau al unui alt domeniu non-global, atunci acea variabilă nu este globală.
Dacă cuvântul cheie var nu este utilizat înainte de un nume de variabilă, atunci acea variabilă este globală.
